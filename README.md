# Smart Hub Service

An AI-powered agency and futuristic startup providing innovative smart service solutions.

## Overview

Smart Hub Service is a modern, responsive web platform designed to showcase AI-driven services and solutions. The project features a sleek, contemporary interface built with Bootstrap and custom styling.

## Features

- **Responsive Design**: Fully responsive layout that works seamlessly across all devices
- **Modern UI/UX**: Built with Bootstrap and custom CSS for a contemporary look
- **Smooth Animations**: Integrated AOS (Animate On Scroll) for elegant page animations
- **Icon Integration**: Font Awesome 4.7.0 for comprehensive icon support
- **Professional Branding**: Consistent brand elements and visual identity

## Getting Started

### Prerequisites

- A web browser (Chrome, Firefox, Safari, Edge)
- A local or remote web server

### Installation

1. Clone the repository:
```bash
git clone https://github.com/yourusername/smarthubservice.git
```

2. Navigate to the project directory:
```bash
cd smarthubservice
```

3. Open `index.html` in your web browser or serve through a local server:
```bash
# Using Python 3
python -m http.server 8000

# Using Node.js http-server
npx http-server
```

4. Visit `http://localhost:8000` in your browser

## Configuration

### Custom Domain

Edit the `CNAME` file to set up a custom domain for GitHub Pages deployment.

### Styling

- **Global Styles**: Modify `css/custom.css` for site-wide styling changes
- **Bootstrap Customization**: Override Bootstrap classes in `css/custom.css`
- **Responsive Breakpoints**: Bootstrap's default breakpoints are included

## Deployment

This project is configured for GitHub Pages deployment. The `CNAME` file enables custom domain support.

### Deploy to GitHub Pages

1. Push your changes to the main branch
2. GitHub Pages will automatically build and deploy from the `index.html` file
3. Visit your site at `https://yourdomain.com` (or the configured custom domain)
